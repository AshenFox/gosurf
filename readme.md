# GoSurf

Интерактивная лендинг страница интернет магазина для серфинга.

## О проекте

Современная лендинг страница с множеством анимированных элементов, слайдерами, меню. Собрана с помощью Webpack.

## Технологии

- HTML
- SCSS
- Webpack
- Animate.css
- Jquery
- JavaScript
